# Mxolisi-Ngema-Web-Developer
Portfolio website of Mxolisi Ngema showcasing web projects, skills, and contact information.

## Deploying to Vercel ✅
You can deploy this Next.js project to [Vercel](https://vercel.com) (recommended).

- Option A — Git Integration (recommended):
  1. Push this repository to GitHub, GitLab, or Bitbucket.
  2. In Vercel, choose **Import Project** and connect your repo.
  3. Use the default build command (`npm run build`) and the framework is detected as Next.js.

- Option B — Vercel CLI:
  1. Install the CLI: `npm i -g vercel`.
  2. Run `vercel` to link the project and `vercel --prod` to deploy.

Notes:
- Ensure any required environment variables are added in your Vercel project settings.
- The project requires Node 18+ (specified in `package.json` under `engines`).
